# Investigation Checklist

Follow these steps when researching a question or exploring an unfamiliar topic.

## Steps

- [ ] **Define the question** — State the specific question or knowledge gap clearly.
- [ ] **Check skills** — Load any relevant skills that might already cover the topic.
- [ ] **Check memories** — Review memories for prior research or related findings.
- [ ] **Gather information** — Collect data from code, docs, logs, or other available sources.
- [ ] **Analyse** — Synthesise findings and draw conclusions supported by evidence.
- [ ] **Present** — Structure the answer clearly with key takeaways and supporting detail.
- [ ] **Record memory** — Save a memory entry with the findings so future investigations can build on them.
